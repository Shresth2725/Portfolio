# Shresth | DevOS Portfolio

An interactive, terminal-inspired personal portfolio designed for a unique, developer-centric experience. This project transitions from a traditional linear scroll to a dynamic, bento-box widget grid layout—resembling a developer's operating system (DevOS).

## 🚀 Overview

This portfolio showcases technical skills through a modern, premium aesthetic. It features:
- **Interactive Terminal theme**: A boot-sequence inspired UI with terminal-style interactions.
- **Bento Box Layout**: A grid of widgets providing quick insights into profile, skills, projects, and activity.
- **GitHub Metrics**: Real-time visualization of coding activity and contributions.
- **Project Showcase**: A dedicated section to explore key development projects.
- **Responsive Design**: Fluid experience across all devices with smooth animations and neon accents.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **UI Library**: [React 19](https://reactjs.org/)
- **Styling**: [Styled Components](https://styled-components.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [Vercel](https://vercel.com/) (Recommended)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Shresth2725/Portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio_website
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Local Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
